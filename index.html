How Auto-Redirects Work:

By placing the `window.location.href = deepLinkUrl;` line directly within the `DOMContentLoaded` event, the browser is instructed to navigate to the deep link as soon as the HTML content is fully loaded and parsed. This initiates the deep link process immediately without any user interaction.

  • This is ideal for "bridge pages" that are solely meant to redirect users.
  • Consider user experience: sometimes a brief message like "Redirecting..." is helpful before the auto-redirect.
